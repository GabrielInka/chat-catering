# Menú Baby Shower

## Descripción

Organiza tu Baby Shower con nosotros!!!, Delicioso menú  con una combinación de canapés fríos, calientes y un postre, ideal para el aperitivo de una tarde inolvidable. Todos los quesos de nuestro menú Baby Shower están pasteurizados. ¿Te gustan las cup cakes de la foto? podemos personalizarlas a tu gusto. llámanos para mayor información.
