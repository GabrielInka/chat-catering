# Menú Entre Amigos

**Identificador:** `menu-entre-amigos`

**Enlace de pedido:** https://www.cateringencasa.com/pedir/menu-entre-amigos/

## Descripción

Delicioso menú con una combinación de canapés fríos, calientes y un postre, ideal para comidas, cenas o para un picoteo.

## Notas de contexto (interno / web)

- Sobre comida o picoteo
- ¿Cuandos canapés necesito?
- Los canapés no son Pinchos Vascos, se comen de un solo bocado.
- Para que te hagas una idea, para comer a base de esta oferta tienes que calcular entre 22 y 24 canapés por persona. Si has escogido un picoteo serán entre 12-14 canapés por persona.
- Los canapés los entregamos en las cajitas  (dependiendo disponibilidad) perfectamente etiquetados con las intolerancias e ingredientes alérgenos. Si no has contratado nuestras bandejas extra para el montaje puedes pasarlo a platos que tengas en casa o presentarlos en las cajitas tal cual. Dependerá del toque de formalidad que le quieras dar a tu Evento
