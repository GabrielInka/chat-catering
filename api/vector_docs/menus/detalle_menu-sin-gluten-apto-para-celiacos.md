# Menú sin gluten (apto para celiacos)

**Identificador:** `menu-sin-gluten-apto-para-celiacos`

**Enlace de pedido:** https://www.cateringencasa.com/pedir/menu-sin-gluten/

## Descripción

Delicioso menú Sin Gluten con una combinación de canapés fríos, calientes y un postre, ideal para comidas, cenas o para un picoteo apto para Celíacos. Ahora puedes disfrutar de la mejor comida para celiacos (sin gluten) a domicilio.

## Alérgenos por producto (resumen en datos)

- **QUESO SEMI CURADO CON UVAS:** Lactosa
- **CROQUETAS DE JAMÓN SERRANO:** Lactosa
- **BROCHETAS DE CHERRY, MOZZARELLA Y ACEITUNAS:** Lactosa
- **BOMBONES DE Q DE CABRA CON HIGOS O MEMBRILLO:** Leche-Lactosa-Lisozima de huevo
- **MINI PIZZAS BARÚ:** Puede contener trazas de Huevo, Leche, Alcohol y Lisozimas de queso
- **CRUJIENTE DE PARMESANO CON DÁTIL Y LOMO:** Lactosa, lisozima de Huevo y Frutos Secos
- **ENSALADITA DE GULAS:** Huevo, Pescado, Soja, Molusco y Crustáceos
- **NACHOS CON GUACAMOLE CASERO:** Ninguna intolerancia
- **MERLUZA REBOZADA CON LIMÓN:** Pescado y Huevo
- **HUEVOS ESCOCESES:** Huevo, Mostaza, Sulfitos y Pescado
- **NUGGETS DE POLLO CON MOSTAZA DULCE:** Huevo. La Salsa: Mostaza, Soja y Apio
- **BROCHETAS DE FRUTA:** Kiwi, Melocotón y Fresa o Uvas
